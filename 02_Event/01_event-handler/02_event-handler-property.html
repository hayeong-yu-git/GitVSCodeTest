02. 이벤트 핸들러 프로퍼티 방식

        window객체와 Document, HTMLElement타입의 DOM노드객체는 이벤트에 대응하는 이벤트핸들러 프로퍼티를 갖고있다
        이벤트핸들러 프로퍼티의 키는 이벤트핸들러 어트리뷰트와 동일하며(on접두사 + 이벤트타입)
        이벤트핸들러 프로퍼티에 함수를 바인딩하면 이벤트핸들러가 등록된다

        이벤트핸들러 어트리뷰트 방식과 비교했을때 HTML과 자바스크립트가 뒤섞이는 문제는 해결할수있지만
        이벤트핸들러 프로퍼티에 하나의 이벤트핸들러만 바인딩할수있다는 단점이 있다
