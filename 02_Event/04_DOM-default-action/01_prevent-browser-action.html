01. 브라우저 기본 동작 중단

        이벤트 객체의 preventDefault메서드(이벤트 막는 메서드)는 DOM요소의 기본동작을 중단시킨다
        예를들어 a요소를 클릭하면 href어트리뷰트에 지정된 링크로 이동하고,
        checkbox 또는 radio요소를 클릭하면 체크 또는 해체되는 것 등을 기본동작이라고 한다
        
        이벤트를 막아야 하는 경우란?
         폼태그를 통해서 값을 서버에 전달하는데,
        만약에 회원가입에서 사용자 입력값의 유효성이 틀렸다면 submit버튼 눌렀을때 
        (틀렸음을 alert띄우는 것과 별개로) 잘못된 값을 들고 가면 안되므로 막아줘야함
클릭해도 절대 이동할 수 없는 a태그 클릭해도 절대 체크되지 않는 체크박스
