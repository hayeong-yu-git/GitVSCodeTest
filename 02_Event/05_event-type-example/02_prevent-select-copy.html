02. 선택막기, 복사막기

        mousedown, mousemove 이벤트가 발생할때 나타나는 브라우저 기본 동작을 막으면 글씨 선택을 막을수 있다
        copy이벤트가 발생할때 나타나는 기본 동작을 막아야 복사(ctrl+C했을때 클립보드에 저장되는일)를 막을수도 있다

        브라우저 기본 동작을 막는 방법
        1. 이벤트 객체의 preventDefault메서드 호출
        2. 이벤트 핸들러 함수 반환값을 false로 지정
이 영역은 드래그해도, 더블클릭해도 선택되지 않도록 합니다
